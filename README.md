# Nettsider for Advokatfirmaet Auxilium

##  Prosjektbeskrivelse

Dette er et moderne, dynamisk nettsted utviklet for **Advokatfirmaet Auxilium** som erstatning for deres eksisterende statiske nettside. Prosjektet transformerer en tradisjonell, innholdstung side til en rask, responsiv og brukervennlig opplevelse.

##  Nye Funksjoner

-  **Rask lastetid** - Bygd med Vite og optimaliserte ressurser
-  **Responsiv design** - Fungerer perfekt på mobil, nettbrett og desktop
-  **Moderne UI** - Bruker Tailwind CSS for profesjonelt utseende
-  **Tilgjengelighet** - WCAG-kompatible fargekontraster og navigasjon
-  **SEO-vennlig** - Strukturert med semantisk HTML

---

##  Teknologier

- **Frontend:** React 19 + TypeScript
- **Bygg-verktøy:** Vite
- **Styling:** Tailwind CSS v4
- **Ikoner:** Lucide React
- **UI Komponenter:** Radix UI
- **Deploy:** GitHub Pages + GitHub Actions
